String class

Strings are objects that represent sequences of characters.


Note that this class handles bytes independently of the encoding used: If used to handle sequences of multi-byte or variable-length characters (such as UTF-8), all members of this class (such as size), as well as operator[], will still operate in terms of bytes (not actual encoded characters).
